Nansi simplifies working with the xterm-256 color palette. Originally designed for Shell usage, Nansi has since been adapted for the web and formatted for software supporting the ASE (Adobe Swatch Exchange) format. You can download or browse the source on Github.

REFERENCE

The keyboard numbers 1 through 4 will toggle the following options. Switch bewteen a 1 dark/light background, 2 reference/hex values, 3 small/large swatch size, and turn column spacing 4 off/on, respectively.

If you have the nansi.sh file installed (see below), you can print the reference palette to standard output with the following command:

N256

INSTALL

Source into your .bashrc or dotfile of choice, import the partial into your Sass/Less stylesheet, or load into your Adobe Swatch Panel, or any program supporting the ASE (Adobe Swatch Exchange) format.

source ~/.nansi.sh
@import “_nansi.hex.scss”;
Swatches Panel → Menu → Load Swatches → nansi.ase

SH/BASH

The syntax is straight forward; f0–255 to set a foreground color, and b0–255 to set a background color; respectively. To set text bold use: bold. To set reversed text (foreground becomes background), use rvrs. To underline text use undr, and finally to end a color/style use stop.

set the foreground then background
printf "${F42}hamburg${B89}fonstiv${STOP}"
set the foreground and background
printf "${F42}${B89}hamburgfonstiv${STOP}"
set multiple foreground colors
printf "${F124}hamb${F126}urg${F128}fontstiv${STOP}"

SCSS + LESS

Nansi is composed of basic color variables, no matter which version you use (hex, hsl, rgb), or which pre-processor Sass or Less. For Sass it's the dollar sign, and for Less, the at sign.

sass/scss = $_0–255
h4 { color: $_32; border-bottom: 1px solid $_32; }
less = @_0–255
h5 { color: @_133; border: 1px solid @_96; }
